# Bank-Loan-Analysis-SQL-PowerBi-
## Project Overview

This project analyzes bank loan data to track key performance indicators (KPIs) such as total loan applications, approved loans, rejected loans, and total disbursed amount.
The goal is to help financial teams monitor loan performance and identify potential risk patterns.

 ## Tools & Technologies

SQL – Data extraction, cleaning, and aggregation

Power BI – Dashboard creation and data visualization

Excel / CSV – Raw data handling

## Key Features

Interactive Power BI dashboard showing:

Total Loan Applications, Funded Amount, Average Interest Rate, and DTI
Loan Status breakdown (Approved, Charged Off, Current, etc.)
Trends by Month, Purpose, and Grade

## SQL queries for:
Data cleaning and preprocessing

Creating views for Power BI connection
